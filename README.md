To download using git:
------------------------
git clone https://github.com/tojb/tumbling_cross_section.git

-----------------------
To compile on Linux:
-----------------------
cd src;
make

-----------------------
To install on Linux:
-----------------------
cd src;
make;
sudo make install

-----------------------
To compile on Windows:
-----------------------

--Install minGW (a free C compiler)

run src/compile_minGW.bat
